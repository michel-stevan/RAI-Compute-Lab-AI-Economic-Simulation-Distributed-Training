# RAI Compute Lab
AI Economic Simulation & Distributed Training Sandbox on RAI Network

## Overview
RAI Compute Lab is an experimental AI sandbox that simulates economic systems using agent-based modeling and trains predictive neural networks using distributed compute.

This project demonstrates:
- AI Training
- AI Inference
- Economic Simulation
- Distributed Compute Integration (RAI Testnet)

## Why This Project?
The future of AI infrastructure is decentralized compute. This project explores how distributed compute networks like RAI can power:
- Economic simulations
- Predictive modeling
- Scalable AI training workloads

## Core Features
- Agent-based economic simulation
- Wealth distribution modeling
- Gini coefficient tracking
- Neural network training (PyTorch)
- Inference API endpoint
- Designed for RAI compute integration

## Architecture
User → FastAPI Backend → RAI Compute → Model → Results Dashboard

See docs/architecture.md for detailed explanation.
